=================
API Documentation
=================

The classes and methods in *sick* are described on this page. The ``sick.models.Model`` class generates data, and the ``sick.analysis`` class 
performs the probabilistic modelling. Post-processing is performed using the ``sick.plot`` module, and general functions are contained in the
``sick.utils`` and ``sick.specutils`` modules.
